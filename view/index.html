Vision

Enable the construction of rich websites, webapps and webservices with re-usable components using nothing but vanilla HTML and CSS by default. Provide a well designed web publishing toolkit, with powerful technologies under the hood rolled into an incredibly intuitive, simple interface. Eradicate the battle with server side languages by just neatly side-stepping them. Is there a bigger user community than the HTML user community ? It's time to let these users build with HTML !

Righteous.


Overview

Silk will evolve so it can be used in two ways, the first is to build simple static websites, the second to build dynamic webapps and webservices. The static website version is in its infancy, but useful enough to build with already. We eat our own dogfood, below is a list of sites we have built with Silk.

  • bheap.co.uk
  • this site

Those of you who are itching to get going might want to move on to the getting started section. Otherwise read on for some information on features.


Features

  • Build re-usable components lego kit style
  • Build static sites in a dynamic way
  • Use nothing but HTML, CSS if you like
  • Productive intuitive CLI (Command Line Interface)
